# Contributing

Thanks for suggesting a tool. This collection is curated, so not everything gets added — but good suggestions are always welcome.

## What belongs here

- AI skills, MCP servers, configs, or workflow tools
- Useful for people who create content, do knowledge work, or ship ideas
- Actually works — you or someone you trust has used it in a real workflow

## What doesn't belong

- Theoretical frameworks or research papers (those go in the learning repo)
- Tools that require deep engineering setup with no creator-facing payoff
- Anything that's just a wrapper around a single API call

## How to suggest

**Option 1: Open an issue**
- Name the tool
- Link to it
- One sentence on why it's worth including
- Mention which section it fits

**Option 2: Submit a PR**
- Add the tool to the correct table in README.md
- Follow the existing format (Name | What it does | Author)
- Link to the source repo, not a fork

## Questions?

Open an issue or reach out directly.
